Memory Limit
512 MB

The maximum amount of RAM the honeypot container can use. Allocating sufficient memory ensures the service runs properly.

Default: 512 MB (recommended for most honeypot types)

Range: 64MB - 1024MB
Memory Swap Limit
RAM
512 MB
SWAP
0 MB
Total:
512 MB

The combined RAM and swap space available to the container. When RAM is filled, the system can use swap space (slower disk-based memory).

Setting this equal to Memory Limit disables swap usage.

SWAP = Memory Swap − Memory
Range: 64MB - 2048MB (Must be ≥ Memory Limit)
